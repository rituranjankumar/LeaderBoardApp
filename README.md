#   Leaderboard Full Stack App

A full-stack app to track users and award random points with dynamic leaderboard ranking.

##  Tech Stack

- React.js + Vite (Frontend)
- Tailwind CSS
- Node.js + Express (Backend)
- MongoDB + Mongoose
